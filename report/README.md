# Report Folder Guide

This folder separates final-facing report materials from historical drafting artifacts.

## Final-Facing Files

| Path | Purpose |
| --- | --- |
| `Group_2_-_Capstone_DataCo_Report_final.docx` | Final academic report document. The same DOCX is submitted through the academic submission system. |
| `final_capstone_report_final_markdown.md` | Markdown source used for the final report package and appendices. |
| `final_artifact_index.md` | Grader/reviewer navigation index for project evidence. |
| `final_validation_summary.md` | Final validation status summary. |
| `final_submission_checklist.md` | Final packaging and manual-submission checklist. |
| `final_report_assets/` | Final report tables, figure sources, dashboard evidence, page inventory, and manifests. |
| `tables/` | Analytical report tables generated by the project workflow and validators. |
| `figures/` | Analytical report figures generated by the project workflow. |

## Dashboard Evidence

Power BI is the finalized dashboard and decision-support layer. The report evidence package includes:

- `final_report_assets/figures/DataCo_Dashboard.pdf`;
- `final_report_assets/figures/Dashboard - Executive view.pdf`;
- `final_report_assets/figures/Dashboard - Command center.pdf`;
- `final_report_assets/figures/figure_2_powerbi_executive_command_center.png`;
- `final_report_assets/figures/powerbi_dashboard_page_inventory.md`;
- `final_report_assets/figures/powerbi_dashboard_page_inventory.csv`.

The `.pbix` file is submitted separately through the academic submission system. The repository stores the Power BI Project source and documentation under `dashboard/`.

## Historical Drafting Files

Superseded drafts, planning notes, APA inventories, dashboard review notes, the earlier generated DOCX conversion output, and obsolete screenshot instructions are under `_drafting/`. These files are retained for audit history and should not be used as final report navigation.

## Final Report Asset Package

The final report asset package is located at:

```text
report/final_report_assets/
```

It contains:

- seven final report tables in Markdown and CSV;
- a Mermaid source file for Figure 1;
- final Power BI dashboard evidence for Figure 2 and appendices;
- source manifests and cleanup documentation.
