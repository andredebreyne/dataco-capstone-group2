## 6. Recommended Report Edit Strategy

The next edit should not update all citations directly yet. The best next step is:

1. Replace only complete citations now: no V2 APA TODO qualifies for `replace_now` because the complete DataCo reference is already cited.
2. Collect missing metadata for the 27 `replace_after_metadata` TODOs.
3. Rewrite or remove the 4 avoidable TODOs:
   - narrow the Chen and Guestrin sentence unless broad XGBoost methodology is retained;
   - remove or rewrite the broad scikit-learn citation sentence unless broad scikit-learn use is discussed;
   - rewrite the responsible-use sentence around internal controls unless a verified governance source is collected;
   - remove the conditional Chen and Guestrin Working References entry if broad methodology is not retained.
4. After metadata collection, perform a citation update pass that replaces keyed APA TODOs with final in-text citations and complete Working References entries.

This sequence avoids fake metadata, keeps analytical conclusions unchanged, and prevents optional sources from entering the final References section unnecessarily.
