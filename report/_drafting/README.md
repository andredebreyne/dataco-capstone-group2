# Report Drafting Artifacts

This folder contains development artifacts used to prepare the final report. These files are retained for traceability, but they are not the primary final-facing report package.

Contents may include:

- earlier report drafts;
- expanded or exploratory report drafts;
- report outlines and gap checklists;
- APA reference planning maps and inventories;
- APA TODO resolution plans;
- selected and verified reference planning files;
- draft change logs.

Final-facing report materials remain in the parent `report/` folder and in `report/final_report_assets/`.

Do not use this folder as the main grader navigation path unless a reviewer specifically needs drafting history.
